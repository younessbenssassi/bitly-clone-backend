# Bitly clone backend

Simple overview of use/purpose.

## Description

This project in a clone of bitly.com.

## Getting Started

### Requirements

* PHP ^8.0
* Laravel Framework 8.83.25

### Installing
#### Composer
```
composer install
```

#### JWT
```
php artisan jwt:secret
```
